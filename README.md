Website for the Auckland University Goju Ryu Karate Club, written in Vue.

Currently hosted at [https://www.uoa-karate.club/](https://www.uoa-karate.club/)

## Running the site

#### Install

 `yarn install`
 
#### Serve
 
 `yarn serve`
 
#### Build 
 
 `yarn build`

#### Test

 `npm test`
